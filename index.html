Estudio de Animación

// Definición de la animación de órbita:

@keyframes orbit { from { transform: rotate(0deg) translateX(80px) rotate(0deg); } to { transform: rotate(360deg) translateX(80px) rotate(-360deg); } }

// Aplicación:

animation: orbit 4s linear infinite;

Efectos Avanzados

Recorte Poligonal: clip-path: polygon(50% 0%, 100% 25%, 100% 75%, 50% 100%, 0%, 75%, 0% 25%);
Recorte Circular: clip-path: circle(50% at 50% 50%);

Nota: Al hacer hover (o click en pantalla táctil), se elimina el clip-path para revelar la imagen completa.
